## AMP8611 - Compact Integrated class AB Stereo Power Amplifier

#### Open hardware project under CERN-OHL-S v2 license
 
#### Amplifier combining a 2-channel input selector, 
#### PGA2311 digital volume control and 45W per-channel LM3886 output stages.
#### Full documentation (description pdf, Kicad project schematic and PCB, STM32 firmware) is available in the repository.
